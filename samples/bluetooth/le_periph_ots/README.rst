.. _bluetooth-periph-ots-sample:

BLE Object Transfer Service Sample
##################################

Overview
********

Application to demonstrate the use of the Object Transfer Service BLE profile.

Requirements
************

* Alif Balletto Development Kit

Building and Running
********************

This sample can be found under :zephyr_file:`samples/bluetooth/le_periph_ots` in the
sdk-alif tree.

When running, the sample application starts advertising and waits for a central to connect.
