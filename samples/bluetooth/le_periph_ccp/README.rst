.. _bluetooth-periph-ccp-sample:

BLE Call Control Profile Server Sample
######################################

Overview
********

Application to demonstrate the use of the Call Control Profile BLE profile.

Requirements
************

* Alif Balletto Development Kit

Building and Running
********************

This sample can be found under :zephyr_file:`samples/bluetooth/le_periph_ccp` in the
sdk-alif tree.

When running, the sample application starts advertising and waits for a central to connect.
