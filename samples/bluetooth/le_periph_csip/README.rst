BLE Coordinated Set Identification Profile Server Sample
########################################################

Overview
********

Application to demonstrate the use of the Coordinated Set Identification Profile (CSIP) BLE profile.

Requirements
************

* Alif Balletto Development Kit

Building and Running
********************

This sample can be found under :zephyr_file:`samples/bluetooth/le_periph_csip` in the
sdk-alif tree.

When running, the sample application starts advertising and waits for a central to connect.
