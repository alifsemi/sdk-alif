BLE Hearing Access Server Sample
##############################################

Overview
********

Application to demonstrate the use of the Hearing Access BLE profile.

Requirements
************

* Alif Balletto Development Kit

Building and Running
********************

This sample can be found under :zephyr_file:`samples/bluetooth/le_periph_haps` in the
sdk-alif tree.

When running, the sample application starts advertising and waits for a central to connect.
